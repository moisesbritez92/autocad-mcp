# Semantic Extraction Pipeline

This repo now has a second-stage semantic pipeline implemented in Python.

## Purpose
It consumes `outputs/plano-contexto.json` generated by the AutoLISP extractor and produces:
- `outputs/plano-contexto.semantic.json`
- `outputs/plano-resumen.md`

## Current capabilities
- heuristic room detection from closed polylines + labels
- opening detection from inserted door/window blocks
- room/opening attachment
- basic room connection inference
- topology graph generation
- architectural validation checks
- agent hints for downstream automation

## Run
```bash
python scripts/semantic_extract.py
```

## Notes
- This stage is heuristic, not a full CAD kernel.
- Best results happen when the base extractor already classified blocks and labels well.
- It is designed to evolve toward the V2.5 / V3 goals in `future_prompts.md`.
