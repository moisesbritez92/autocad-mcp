# Long-term Memory

- El repo contiene `rules/architecture_rules.md`, un documento base de reglas de diseño arquitectónico para agentes CAD/AI.
- Estas reglas deben usarse como referencia para futuras validaciones semánticas del plano, scoring arquitectónico y razonamiento espacial.
- Incluye principios como separación social/privada, eficiencia de circulación, iluminación natural, núcleo húmedo, jerarquía de dormitorios, transición de acceso, alineación estructural y calidad arquitectónica.
