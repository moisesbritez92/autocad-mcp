# 2026-03-16

- Se creó `rules/architecture_rules.md` con reglas de diseño arquitectónico para agentes CAD/AI.
- El documento define heurísticas y validaciones residenciales para razonamiento arquitectónico, incluyendo:
  - separación entre zona social y privada
  - eficiencia de circulación
  - proporciones de habitaciones
  - iluminación natural
  - colocación de puertas
  - agrupación de núcleo húmedo
  - jerarquía de ventanas y dormitorios
  - minimización de espacios residuales
  - alineación estructural
  - espesores típicos de muros
  - transición de acceso / foyer
  - integración de almacenamiento
  - conexión interior-exterior
  - estrategia de ubicación de baños
  - métrica de calidad arquitectónica (`design_score`, `lighting_score`, etc.)
- Debe considerarse como base de reglas para futuras validaciones semánticas del pipeline CAD.
